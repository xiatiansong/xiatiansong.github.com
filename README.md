夏天松的博客
======

[http:/blog.xiatiansong.com](http://blog.xiatiansong.com)博客的源码。

该博客使用 jekyll+bootstrap-3+Markdown 搭建。

使用本主题，请：

- 修改 _config.yaml
- 修改 CNAME
- 删除_post下的所有文件
- 请忽略deploy.sh和 qiniu-images.conf文件的存在
